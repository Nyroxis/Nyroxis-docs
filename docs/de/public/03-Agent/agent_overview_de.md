# Agent – Übersicht

Der Nyroxis Agent ist die zentrale Komponente, die auf dem Gerät des Benutzers ausgeführt wird und alle sicherheitsrelevanten Aktivitäten lokal überwacht, analysiert und verschlüsselt speichert.  
Er wurde so entwickelt, dass er leichtgewichtig, offline-fähig und vollständig datenschutzorientiert arbeitet.

## Aufgabenbereich
Der Agent ist für folgende Kernfunktionen verantwortlich:
- Erfassung von System-, Prozess-, Netzwerk- und Dateiaktivitäten  
- Sofortige Verschlüsselung aller erfassten Ereignisse  
- Schutz der Integrität der gespeicherten Daten  
- Übermittlung relevanter Daten an die lokale Analyse-Engine  
- Bereitstellung einer manipulationsresistenten Überwachungsgrundlage  

## Eigenschaften
- Minimaler Ressourcenverbrauch  
- Vollständig lokal, keine Cloud-Abhängigkeit  
- Robust und stabil im Dauerbetrieb
