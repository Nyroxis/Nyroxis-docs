# Dashboard UI Design

Das Nyroxis Dashboard bietet sowohl nicht-technischen Nutzern als auch Fachleuten eine klare, intuitive und handlungsorientierte Sicht auf den Sicherheitszustand ihres Geräts.  
Das Design folgt Prinzipien von Klarheit, Minimalismus, Privatsphäre und Echtzeitwahrnehmung.

## Designphilosophie

### **1. Klarheit vor Komplexität**
Sicherheitsdaten sollten den Nutzer nicht überfordern.  
Das Dashboard verwandelt rohe Logs in verständliche Erkenntnisse durch:
- klare Typografie  
- übersichtliche Abstände  
- neutrale Symbole  
- farbcodierte Schweregrade  

### **2. Privacy-First Visualisierung**
Alle angezeigten Daten:
- stammen aus lokal verschlüsseltem Speicher  
- verlassen niemals das Gerät  
- werden vollständig offline verarbeitet  

### **3. Konsistente UX für alle Personas**
Die Oberfläche unterstützt:
- Familien  
- Fachleute  
- Führungskräfte  
- Cybersicherheitsanalysten  

## Zentrale UI-Bereiche

### **1. Kopfzeile**
Beinhaltet:
- Nyroxis Identität  
- Navigation  
- Einstellungen & Sprache  

### **2. Linkes Navigationspanel**
Strukturierte Navigation für:

- Events & Suche  
- Detections  
- Correlations  
- Scenarios  
- AI Nyroxis (NyXIA)  
- Reports  
- Severity Manager  
- Rule Matching Builder  
- Settings  
- Help / Documentation  
- About Nyroxis  

### **3. Arbeitsbereich**
Der aktive Bereich mit:
- Zeitachsen  
- Tabellen  
- Kontextpanels  
- Diagrammen  
- Rule Builder  

### **4. Fußzeile / Statuslinie**
Zeigt:
- Agent-Status  
- letzten Datenbankzugriff  
- Ressourcenverbrauch (optional)  

## Visueller Stil

### Farben
- Blau/Cyan: informativ  
- Gelb: Warnung  
- Rot: kritisch  
- Grün: normal  

### Typografie
Moderne, gut lesbare Schriftarten mit klarer Hierarchie.

### Symbole & Indikatoren
Einfache, neutrale Symbole zur besseren Orientierung.

## Responsivität
Optimiert für Desktop, aber skalierbar auf:
- Laptops  
- HiDPI  
- Ultrawide  

## Zusammenfassung
Das Nyroxis Dashboard UI verbindet Klarheit, Privatsphäre und Benutzerfreundlichkeit — damit jeder Nutzer seinen Sicherheitsstatus schnell verstehen kann.
