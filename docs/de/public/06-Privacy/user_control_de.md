# Benutzerkontrolle & Transparenz

Nyroxis wurde entwickelt, um Nutzern vollständige Kontrolle über ihre Sicherheitsdaten, Systemverhalten und Datenschutzeinstellungen zu geben.  
Keine versteckten Prozesse, keine Hintergrunduploads, keine Cloud — alles ist transparent und lokal.

## Grundprinzipien der Benutzerkontrolle
Nyroxis ermöglicht Nutzern:
- zu sehen, was überwacht wird  
- zu kontrollieren, was gespeichert wird  
- Daten jederzeit zurückzusetzen oder zu löschen  
- die Offline‑Arbeitsweise zu verifizieren  
- KI‑Entscheidungen zu verstehen  

Der Benutzer bleibt immer vollständig in Kontrolle.

## 1. Keine stille Datensammlung
Nyroxis erfasst **nur**, was für Sicherheitsanalysen notwendig ist:
- Prozesse  
- Netzwerkevents  
- Dateiänderungen  
- Berechtigungsaktionen  

Keine persönlichen Dateien oder Inhalte.

## 2. Vollständiger Reset
Benutzer können zurücksetzen:
- Logs  
- Metadaten  
- KI‑Baseline  
- Korrelationscache  

Der Reset stellt ein vollständig sauberes System her.

## 3. Lokale Speicherung
Alle Daten sind:
- lokal gespeichert  
- vollständig verschlüsselt  
- niemals extern übertragen  

Keine Cloud‑Konten, keine Remote‑Abhängigkeiten.

## 4. Transparente KI
NyXIA bietet:
- klare Erklärungen  
- Begründungen für Entscheidungen  
- Szenario‑Zusammenfassungen  

Keine Black‑Box‑Logik.

## 5. Anpassungsmöglichkeiten
Benutzer können konfigurieren:
- Aufbewahrungszeit  
- Schwellwerte für Schweregrad  
- Regelpräferenzen  
- Benachrichtigungen  

Alle Einstellungen sind optional und lokal.

## 6. Überprüfung der Offline‑Arbeitsweise
Nyroxis ermöglicht die Überprüfung, dass:
- keine externen Verbindungen bestehen  
- keine Telemetrie vorhanden ist  
- keine Cloud genutzt wird  

Vertrauen ist überprüfbar.

## Zusammenfassung
Nyroxis gibt Nutzern vollständige Kontrolle über Daten, Transparenz der Abläufe und die Möglichkeit, alles lokal zu verwalten — ohne Cloud, ohne versteckte Prozesse.
