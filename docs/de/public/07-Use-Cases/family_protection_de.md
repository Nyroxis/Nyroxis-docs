# Familienschutz

Nyroxis schützt Familiengeräte vollständig offline – ohne Inhalte zu überwachen oder persönliche Daten auszulesen.

## Warum Familienschutz wichtig ist
Moderne Risiken:
- stille Malware  
- unsichere Downloads  
- Phishing  
- kompromittierte Heimgeräte  
- Stalkerware  

## 1. Lokale, private Sicherheit
Schutz für:
- Kindergeräte  
- Elternlaptops  
- gemeinsame Familiencomputer  

Alle Daten bleiben zu Hause.

## 2. Frühe Risikoerkennung
Nyroxis erkennt:
- gefährliche Downloads  
- unbekannte Prozesse  
- unerwartete Netzwerkevents  
- Muster ähnlich Ransomware  

## 3. Offline‑KI für Haushaltssicherheit
NyXIA:
- erkennt Anomalien  
- erklärt Szenarien  
- liefert Frühwarnsignale  

## 4. Vollständige Wahrung der Privatsphäre
Nyroxis überwacht niemals:
- Browserverlauf  
- Dateien  
- Fotos  
- Chats  

Nur sicherheitsrelevante Events.

## 5. Ein Modell für mehrere Benutzer
Jedes Gerät erstellt ein eigenes lokales Profil.

## Zusammenfassung
Nyroxis schützt Familiengeräte sicher und privat – ohne Cloud und ohne Inhaltsüberwachung.
