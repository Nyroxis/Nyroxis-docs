# Use Cases – Übersicht

Nyroxis schützt Privatpersonen, Familien, Fachkräfte und Nutzer mit hohem Risiko – vollständig offline und privat.

## 1. Persönliche Sicherheit
Nyroxis:
- erkennt verdächtiges Verhalten  
- markiert ungewöhnliche Aktivität  
- arbeitet offline  

## 2. Familienschutz
Nyroxis:
- schützt Kinder‑ und Elterngeräte  
- erkennt unsichere Muster  
- speichert alles lokal  

## 3. Führungskräfte & Hochrisikoziele
Nyroxis:
- analysiert Verhalten offline  
- sendet keine Daten  
- bietet frühe Warnsignale  

## 4. Fachkräfte in sensiblen Bereichen
Schutz für:
- Anwälte  
- Journalisten  
- Psychologen  
- NGOs  

## 5. Auftragnehmer & Berater
Für Geräte ohne Unternehmens‑EDR:
- lokale Sicherheit  
- keine Überwachung  
- offline‑Analyse  

## 6. Technische Nutzer
Nyroxis bietet:
- Rohlogs  
- Transparenz  
- Offline‑Analyse  

## Zusammenfassung
Nyroxis deckt vielfältige Anwendungsfälle ab — privat, verschlüsselt und vollständig lokal.
