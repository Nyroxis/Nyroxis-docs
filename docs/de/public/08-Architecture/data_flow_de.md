# Datenfluss

Der Datenfluss in Nyroxis ist klar strukturiert und vollständig lokal.  
Alle sicherheitsrelevanten Ereignisse werden gesammelt, verschlüsselt, analysiert und visualisiert — ohne jemals das Gerät zu verlassen.

## 1. Erfassung
Der Nyroxis Agent sammelt:
- Prozessaktivität  
- Dateioperationen  
- Netzwerkanfragen  
- Berechtigungsänderungen  
- Systemereignisse  

Alles wird in Echtzeit erfasst.

## 2. Verschlüsselung
Vor der Speicherung werden Daten:
- sofort verschlüsselt  
- niemals im Klartext abgelegt  
- durch verknüpfte Hash-Blöcke geschützt  

## 3. Speicherung
Die verschlüsselten Ereignisse werden in der lokalen Datenbank abgelegt:
- offline  
- manipulationsresistent  
- vollständig privat  

## 4. Analyse
Die Analyse erfolgt durch:
- die lokale Analyse-Engine  
- das KI-Modul NyXIA  
- Rule Matching  
- Korrelationen  

Ziel ist das Erkennen von:
- Anomalien  
- Sequenzen  
- Risiken  
- Mustern  

## 5. Visualisierung
Das Dashboard zeigt:
- Logs  
- Diagramme  
- Schweregrade  
- Szenarien  
- KI-Erklärungen  

## 6. Kontrolle durch den Benutzer
Der Benutzer kann jederzeit:
- Daten zurücksetzen  
- Ereignisse durchsuchen  
- Regeln konfigurieren  
- Offline-Betrieb überprüfen  

## Zusammenfassung
Der Datenfluss in Nyroxis stellt sicher, dass jede Phase — von Erfassung bis Analyse — privat, lokal und verschlüsselt bleibt.
