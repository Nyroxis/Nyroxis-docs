# Verschlüsselung

Nyroxis verwendet ein vollständig lokales, privacy-first Verschlüsselungsmodell.  
Jedes Ereignis wird sofort beim Erfassen verschlüsselt, und keine Daten verlassen jemals das Gerät.

## Warum Verschlüsselung wichtig ist
Sicherheitsdaten können sensible Informationen enthalten – deshalb muss die Speicherung:
- privat  
- lokal  
- geschützt gegen Manipulation  
- unabhängig von der Cloud  

sein.

## Wie Nyroxis verschlüsselt
Nyroxis verschlüsselt Daten:
- direkt bei der Erfassung  
- bevor sie gespeichert werden  
- mit einem gerätegebundenen Schlüssel  
- ohne Klartextspeicherung  

### Eigenschaften der Verschlüsselung
- vollständig lokal  
- kein Upload  
- kein Remote-Zugriff  
- Schlüssel werden nicht im Code hinterlegt  

## Schutz der Integrität
Zusätzlich nutzt Nyroxis:
- Hash-Verkettung  
- Integritätsprüfungen  
- sichere Schreibvorgänge  

Dies verhindert das Verändern oder Löschen von Ereignissen.

## Zusammenfassung
Nyroxis stellt sicher, dass alle Daten privat, verschlüsselt und nur auf dem Gerät des Benutzers zugänglich bleiben.
