# Lokale KI

Nyroxis verwendet eine vollständig lokale KI-Engine (NyXIA), die sicherheitsrelevante Ereignisse analysiert, ohne jemals Daten an einen Server zu senden.

## Warum lokale KI?
Die KI soll helfen:
- ungewöhnliches Verhalten zu erkennen  
- Muster sichtbar zu machen  
- Risiken besser einzuschätzen  

ohne Privatsphäre zu gefährden.

## Was lokale KI bedeutet
Lokale KI bedeutet:
- keine Cloud  
- keine Telemetrie  
- keine Online-Analyse  
- alles bleibt auf dem Gerät  

## Was NyXIA analysiert
NyXIA bewertet:
- Prozessketten  
- Netzwerkverhalten  
- Dateiaktivität  
- zeitliche Muster  

und erzeugt:
- Anomaliewerte  
- Erklärungen  
- Szenarien  

## Vorteile
- funktioniert offline  
- schützt die Privatsphäre  
- geringe Systemlast  
- schnelle Analyse  

## Zusammenfassung
Die lokale KI von Nyroxis stärkt Sicherheit – ohne Daten offenzulegen.
