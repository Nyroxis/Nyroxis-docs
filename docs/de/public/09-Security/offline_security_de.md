# Offline-Sicherheit

Nyroxis wurde so entwickelt, dass es vollständig offline betrieben werden kann – ohne Cloud, ohne Telemetrie und ohne Internetzugang.

## Warum Offline wichtig ist
Ein echtes Sicherheitsprodukt muss unabhängig sein von:
- externen Servern  
- Online-Verifizierung  
- Internetqualität  

Offline bedeutet:
- geringere Angriffsfläche  
- keine Datenübertragung  
- mehr Kontrolle für den Nutzer  

## Was offline funktioniert
Alles in Nyroxis funktioniert offline:
- Ereigniserfassung  
- Verschlüsselung  
- Speicherung  
- lokale KI  
- Rule Matching  
- Detections  
- Szenarien  
- Dashboard  

## Updates (optional offline)
Updates können:
- manuell  
- über Offline-Pakete  

installiert werden.

## Zusammenfassung
Nyroxis bietet vollständige Sicherheit ohne Internet – privat, lokal und verlässlich.
