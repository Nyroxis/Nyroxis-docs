# Sicherheitsübersicht

Nyroxis bietet Sicherheit auf Enterprise-Niveau, jedoch speziell für persönliche Geräte, Familienumgebungen und unabhängige Fachkräfte – ohne Cloud-Abhängigkeit.

## Kernprinzipien

### Privacy-First
- keine Telemetrie  
- keine Inhaltsanalyse  
- keine Cloud  

### Lokal
- Speicherung  
- Verschlüsselung  
- Analyse  
- KI  
- Korrelationen  

alles findet auf dem Gerät statt.

### Transparenz
Nutzer sehen:
- was erfasst wird  
- wie es verarbeitet wird  
- wie Entscheidungen entstehen  

## Wichtige Sicherheitsfunktionen
Nyroxis bietet:
- sofortige Verschlüsselung  
- manipulationsresistente Speicherung  
- lokale KI-Erkennung  
- Offline-Betrieb  
- klare Erklärungen für Risiken  

## Keine Überwachung
Nyroxis überwacht NICHT:
- Dokumente  
- Fotos  
- Nachrichten  
- Browserverlauf  

Es verarbeitet ausschließlich sicherheitsrelevante technische Ereignisse.

## Zusammenfassung
Nyroxis bietet starke Sicherheit mit maximaler Privatsphäre – ideal für persönliche und berufliche Geräte.
