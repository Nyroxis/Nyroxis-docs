# Manipulationsschutz

Nyroxis wurde so entwickelt, dass ein Angreifer Ereignisdaten weder verändern noch löschen kann.  
Das Ziel ist, die Integrität der Sicherheitsinformationen jederzeit sicherzustellen.

## Warum Manipulationsschutz wichtig ist
Ein Angreifer versucht oft:
- Spuren zu löschen  
- Logs zu verändern  
- Einträge zu überschreiben  
- Aktivitäten zu verstecken  

Nyroxis verhindert das.

## Wie Nyroxis Manipulation verhindert
Nyroxis schützt Ereignisdaten durch:
- hashverkettete Blöcke  
- Integritätsprüfungen  
- sichere Schreibpfade  
- geschützte Datenbankstruktur  
- Anti-Tamper-Logik beim Lesen  

## Keine Klartextspeicherung
- Ereignisse werden niemals im Klartext geschrieben  
- Änderungen wären sofort erkennbar  

## Zusammenfassung
Der Manipulationsschutz garantiert, dass sicherheitsrelevante Daten zuverlässig und unverfälscht bleiben.
