# Key Features

Nyroxis brings enterprise‑style visibility to personal and professional devices, while remaining lightweight, private, and fully offline‑capable.  
Below is a clear overview of the main features that define the Nyroxis platform.

---

## Local Event Monitoring
Nyroxis continuously collects essential security‑relevant activity such as:
- Process creation and termination  
- Network connections  
- File modifications  
- Privilege‑related actions  
- System and security events  

All monitoring happens **locally**, without sending data to the cloud.

---

## Encrypted Local Storage
All collected events are:
- Encrypted at rest  
- Protected against tampering  
- Stored only on the user’s device  

Nyroxis ensures that sensitive activity logs remain private and under user control.

---

## AI‑Assisted Behavioral Analysis
Nyroxis uses lightweight AI models to identify:
- Suspicious behavioral patterns  
- Anomalous activity sequences  
- Early signs of compromise  
- Potential persistence mechanisms  

The AI assists the user by prioritizing relevant events and reducing noise.

---

## Clear, Human‑Friendly Dashboard
The dashboard provides:
- Real‑time event timeline  
- Visual charts and summaries  
- Severity indicators  
- Simple explanations of suspicious actions  

Designed for both non‑technical users and cybersecurity professionals.

---

## Tamper‑Resistant Architecture
Nyroxis includes mechanisms that help protect:
- The agent service  
- Local logs  
- Integrity of event data  

This ensures attackers cannot easily erase their footprints.

---

## Offline‑First Operation
Nyroxis works fully without:
- Cloud connection  
- Internet dependency  
- External authentication  

The platform is designed for high‑privacy environments.

---

## Designed for Real People
Nyroxis fits naturally into:
- Executive workstations  
- Lawyer and doctor laptops  
- Family computers  
- Cybersecurity home labs  
- Remote worker devices  

It provides clarity and confidence without technical complexity.
