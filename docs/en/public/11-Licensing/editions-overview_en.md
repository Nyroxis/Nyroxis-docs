# Editions Overview

Nyroxis is offered in two main editions: **Community** and **Pro**, both built on the same core privacy-first architecture.

---

##  Community Edition
Designed for:
- For evaluation and familiarization with the system  

Includes:
- Agent  
- Dashboard CLI  
- Dashboard GUI  

Completely free.

---

##  Pro Edition
Designed for:
- Executives  
- High-risk individuals  
- Professionals  
- Security-conscious users  

Includes everything in Community plus:
- Enhanced AI (NyXIA Pro)  
- Advanced scenarios  
- Priority protections  
- Deeper visibility and analytics
