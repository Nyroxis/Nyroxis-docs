# Modèle de sécurité

Le modèle de sécurité Nyroxis garantit que toute la collecte, l’analyse et le stockage des événements se font sous des principes stricts de confidentialité et d'intégrité.  
Il est conçu pour les particuliers, professionnels et cadres exigeant une protection fiable sans compromis sur la vie privée.

---

##  Objectifs principaux
Nyroxis protège :
- La confidentialité  
- L’intégrité du système  
- La vie privée de l’utilisateur  
- L’autonomie locale  

---

##  1. Chiffrement local de bout en bout
Tous les événements sont :
- Chiffrés dès la collecte  
- Stockés uniquement sous forme chiffrée  
- Accessibles via des chemins sécurisés  
- Jamais enregistrés en clair  

Les clés sont :
- Locales  
- Dynamiques  
- Non stockées dans l’application  
- Jamais transmises  

---

##  2. Stockage anti-falsification
Nyroxis intègre :
- Vérifications d’intégrité  
- Chemins d’écriture protégés  
- Chaînage de hash  
- Contrôles anti-tamper  

---

##  3. IA privée et locale
NyXIA :
- Fonctionne hors-ligne  
- Analyse les événements chiffrés  
- Produit des détections localement  
- N’envoie jamais de journaux  

---

##  4. Fonctionnement 100% hors-ligne
Nyroxis n’utilise :
- Aucun cloud  
- Aucune API externe  
- Aucune authentification distante  

---

##  5. Rétention minimale
Stockage uniquement :
- Des événements de sécurité  
- Des métadonnées techniques  

Jamais :
- Documents personnels  
- Historique de navigation  
- Données sensibles  

---

##  6. Logique transparente
Modèle volontairement simple :
- Collecte locale  
- Chiffrement local  
- IA locale  
- Alertes locales  

---

##  Résumé
Un modèle moderne et privé :
- Stockage chiffré  
- Anti-tamper  
- IA locale  
- Aucun risque cloud.
