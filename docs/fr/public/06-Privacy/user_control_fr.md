# Contrôle utilisateur & transparence

Nyroxis est conçu pour offrir aux utilisateurs un contrôle total sur leurs données de sécurité, le comportement du système et les paramètres de confidentialité.  
Aucune collecte cachée, aucune transmission cloud — tout est transparent et local.

---

##  Principes fondamentaux
L’utilisateur peut :
- Voir ce qui est surveillé  
- Contrôler ce qui est stocké  
- Réinitialiser ou supprimer ses données  
- Vérifier l’absence de communication réseau  
- Comprendre les décisions de l’IA  

---

##  1. Aucune collecte silencieuse
Nyroxis collecte uniquement :
- Processus  
- Événements réseau  
- Modifications de fichiers  
- Actions de privilège  

Jamais :
- Documents personnels  
- Photos, messages  
- Historique de navigation  

Tout est documenté clairement.

---

##  2. Réinitialisation complète
L’utilisateur peut réinitialiser :
- Journaux  
- Métadonnées  
- Profil comportemental IA  
- Cache des corrélations  

La réinitialisation remet Nyroxis à zéro.

---

##  3. Stockage local uniquement
Toutes les données :
- Restent sur l’appareil  
- Sont chiffrées  
- Ne sont jamais transmises  

Aucun compte cloud ou stockage externe requis.

---

##  4. IA transparente
NyXIA fournit :
- Explications des détections  
- Raison contextuelle  
- Détails des scénarios  

Les décisions IA sont compréhensibles.

---

## ️ 5. Options de personnalisation
L’utilisateur peut régler :
- Durée de conservation  
- Seuils de sévérité  
- Préférences de règles  
- Notifications  

Tout est local.

---

##  6. Vérification du mode hors-ligne
Nyroxis permet de vérifier :
- Absence de connexions sortantes  
- Absence de télémetrie  
- Indépendance totale du cloud  

---

##  Résumé
Nyroxis donne à l’utilisateur un contrôle total, une transparence complète et la possibilité de gérer ou réinitialiser ses données — sans dépendance cloud.
