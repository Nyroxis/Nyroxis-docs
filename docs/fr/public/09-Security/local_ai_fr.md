# IA locale (NyXIA)

NyXIA est le moteur d’IA entièrement local de Nyroxis.  
Il détecte les comportements suspects, analyse les séquences et génère des scénarios — le tout **sans cloud**.

---

##  1. IA 100% hors-ligne
NyXIA ne contacte jamais :
- un serveur externe  
- un cloud  
- une API distante  
- une plateforme de télémétrie  

Toutes les analyses sont locales.

---

##  2. Analyse de séquences comportementales
NyXIA analyse des séquences complètes, permettant de détecter :
- attaques en plusieurs étapes  
- comportements anormaux  
- accès fichiers suspects  
- actions de privilèges inhabituelles  

Plus précis que les analyses événement-par-événement.

---

##  3. Scores d’anomalies
Chaque séquence reçoit :
- un embedding comportemental  
- un score de déviation  
- une classification de risque  
- une explication  

Transparence totale.

---

##  4. Profils locaux
Chaque appareil construit :
- son propre profil normal  
- ses modèles réseau habituels  
- ses schémas d'utilisation  
- ses accès fichiers  

Jamais envoyés en ligne.

---

##  5. Scénarios
NyXIA regroupe les événements en scénarios :
- chaînes de processus  
- anomalies réseau  
- accès fichiers suspects  
- escalades de privilèges  

Chaque scénario comprend :
- description  
- sévérité  
- preuves  
- horodatage  

---

##  6. Aucun entraînement cloud
Rien n’est envoyé :
- journaux  
- profils  
- anomalies  
- données utilisateur  

Tout reste local.

---

## ️ 7. Léger et optimisé
Fonctionne sur :
- ordinateurs personnels  
- postes sensibles  
- environnements isolés  

Sans consommation excessive.

---

##  Résumé
NyXIA fournit :
- IA locale  
- Détection comportementale avancée  
- Confidentialité totale  
- Analyse transparente  

Une IA moderne conçue pour la cybersécurité privée.
