# Sécurité hors-ligne

Nyroxis fonctionne entièrement hors-ligne, sans dépendance à un cloud, serveur externe ou API.  
Cela garantit autonomie, confidentialité et sécurité même en environnement isolé.

---

##  1. Aucune communication cloud
Nyroxis n’envoie jamais :
- De télémétrie  
- De journaux  
- De requêtes externes  
- D’analyses cloud  

Toutes les opérations sont locales.

---

##  2. Gestion locale des clés
Les clés sont :
- Générées localement  
- Liées au matériel  
- Jamais transmises  
- Jamais stockées en clair  

Toute cryptographie est locale.

---

##  3. IA hors-ligne (NyXIA)
NyXIA réalise :
- Détection comportementale  
- Analyse de séquences  
- Scores d’anomalie  
- Scénarios  

Sans cloud ni téléchargement.

---

##  4. Stockage chiffré local
Les journaux restent :
- Chiffrés  
- Stockés en base sécurisée  
- Non transférés  

Aucun risque lié au cloud.

---

##  5. Aucun téléchargement externe
Nyroxis ne télécharge pas :
- Règles  
- Modèles  
- Signatures  
- Threat intel  

Tout est intégré localement.

---

##  6. Compatible environnements isolés
Fonctionne dans :
- Ordinateurs hors réseau  
- Environnements sensibles  
- Zones à risque  
- Secteurs gouvernementaux  

Aucune connexion requise.

---

##  Résumé
Sécurité Nyroxis hors-ligne :
- Autonomie totale  
- Zéro cloud  
- IA locale  
- Données chiffrées  
- Indépendance complète
