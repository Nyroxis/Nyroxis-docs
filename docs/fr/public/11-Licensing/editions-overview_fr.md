# Aperçu des éditions

Nyroxis existe en deux éditions : **Community** et **Pro**, toutes deux basées sur le même modèle de confidentialité totale.

---

##  Community Edition
Pour :
- Pour l'évaluation et la familiarisation avec le système  
 
Inclut :
- Agent  
- Dashboard CLI  
- Dashboard GUI 

Gratuite.

---

##  Pro Edition
Pour :
- Dirigeants  
- Professionnels  
- Utilisateurs à haut risque  

Inclut :
- IA avancée  
- Scénarios enrichis  
- Protection prioritaire  
- Analytique avancée  

---

##  Résumé
Les deux éditions partagent la même base sécurisée — la Pro ajoute des capacités avancées.
