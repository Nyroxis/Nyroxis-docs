# רכיבי הארכיטקטורה

חלק זה מציג את רכיבי הליבה של Nyroxis וכיצד הם משתלבים למערכת אבטחה פרטית, מקומית ועמידה בפני שינוי.

---

##  1. Nyroxis Agent
שירות רקע קל האחראי על:
- ניטור תהליכים  
- מעקב אחר רשת  
- מעקב אחר שינויים בקבצים  
- ניטור פעולות הרשאה  
- איסוף אירועים בזמן אמת  

כל האירועים **מוצפנים מיד**.

יעיל, שקט ואופליין.
