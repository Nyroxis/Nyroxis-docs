# אבטחת אופליין

Nyroxis פועלת באופן מלא ללא אינטרנט — ללא ענן, ללא API חיצוני וללא תלות בשרתים מרוחקים.  
הכול מקומי, פרטי ומוגן.

---

##  1. ללא תקשורת לענן
Nyroxis לעולם אינה שולחת:
- טלמטריה  
- לוגים  
- בקשות לשרתים  
- מידע ל-AI בענן  

הכול נעשה על המכשיר בלבד.

---

##  2. ניהול מפתחות מקומי
המפתחות:
- נוצרים מקומית  
- קשורים לחומרה  
- אינם נשלחים  
- אינם נשמרים כטקסט גלוי  

הצפנה מלאה ללא רשת.

---

##  3. AI אופליין (NyXIA)
NyXIA מבצעת:
- ניתוח התנהגות  
- זיהוי אנומליות  
- ניתוח רצפים  
- יצירת תרחישים  

ללא שרתים חיצוניים.

---

- נשמרים מוצפנים
